# Quick start

## .env

Run `cp example.env .env`

Add your OPEN API KEY to a `.env` file

Add your Assistant to the `.env` file

## Express Server

`npm run start`

create a post request to `/ai` with a body object

```json
{"message":"string"}
```

## CLI Run

`node main.js`
